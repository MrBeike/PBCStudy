## 脚本命名提示
    课程结构：所有项目(program)->单个项目(project)->单个章节(chapter)->单个知识点(knowledge)
    
    学习流程：登录(login)->获取所有项目(getProgram)->获取单个项目(getproject)->获取单个章节(getChapter)->获取单个知识点(getKnowledge)
